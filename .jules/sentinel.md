## 2026-05-10 - Prevent XSS in react-simple-code-editor\n**Vulnerability:** XSS via unescaped user input in react-simple-code-editor when 'highlight' prop doesn't properly sanitize the HTML.\n**Learning:** Never use raw `code => code` for the `highlight` prop in `react-simple-code-editor`, as it uses `dangerouslySetInnerHTML` internally, executing arbitrary scripts when users input HTML/JS payload.\n**Prevention:** Use an `escapeHtml` function to safely encode raw HTML entities (like <, >, &) before returning the highlighted string.
